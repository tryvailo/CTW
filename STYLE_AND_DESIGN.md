# СТИЛИ И ОФОРМЛЕНИЕ САЙТА

## Описание дизайна в техзадании

Согласно `frontend-structure-copy.md`:
- **Design:** Large text (18px+), high contrast, simple navigation
- Минималистичный дизайн с фокусом на доступность
- Без сложных визуальных элементов

## Текущая реализация

### Цветовая схема (WCAG AAA):
- **Фон:** Белый (#ffffff)
- **Текст:** Темно-серый (#1a1a1a) - высокий контраст
- **Основной цвет:** Синий (#0052cc) - для ссылок и кнопок
- **Границы:** Светло-серый (#cccccc)

### Компоненты:
- **Карточки:** Белый фон, серая граница, тень
- **Кнопки:** Синий фон (#0052cc), белый текст, большие размеры (44px+)
- **Таблицы:** Четкие границы, светлый фон для заголовков

### Типографика:
- Body: 18px (больше чем минимум 14-16px)
- H1: 28px
- H2: 24px
- Line height: 1.8 для читаемости

## Возможные улучшения

Если нужен более выразительный дизайн, можно добавить:
1. Градиенты для hero секции
2. Иконки для value props
3. Более яркие акцентные цвета
4. Тени и эффекты для карточек

Но текущий дизайн соответствует требованиям техзадания: **простой, доступный, с большими шрифтами и высоким контрастом**.
